## Запуск проекта

Для запуска проекта необходимо предварительно подготовить рабочее окружение и установить все зависимости.

### 1. Создание виртуальной среды

Виртуальная среда используется для изоляции зависимостей проекта от системного окружения.

```bash
python3 -m venv .venv
```

### 2. Активация виртуальной среды

В зависимости от используемой операционной системы выполните одну из следующих команд.

Linux / macOS:

```bash
source .venv/bin/activate
```

Windows (PowerShell):

```bash
.venv\Scripts\activate
```

### 3. Установка зависимостей

После активации виртуальной среды установите необходимые пакеты из файла requirements.txt.

```bash
pip install -r requirements.txt
```

### 4. Запуск сервисов NATS

Для работы приложения требуется брокер сообщений NATS, который развёрнут с использованием Docker. Запуск сервиса выполняется следующей командой:

```bash
docker-compose up -d
```

### 5. Запуск приложения

После запуска всех необходимых сервисов выполните старт приложения с помощью ASGI-сервера Uvicorn.

```bash
uvicorn app.main:app
```

После выполнения указанных шагов приложение будет доступно и готово к работе.
